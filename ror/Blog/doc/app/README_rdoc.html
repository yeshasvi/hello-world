    <h1>Setup</h1>

    #Windows

    1. Install ruby from here rubyinstaller.org/

    2. Install RVM(Ruby version manager) blog.developwithpassion.com/2012/03/30/installing-rvm-with-cygwin-on-windows/

    3. Use default ruby as 2.1.1

    4. Install Rails 4 gem using cygwin

    • I pprefer to use cygwin for setting up ruby dev enviorement

    #Ubuntu or #Mac

    By default ruby comes on ubuntu and mac or follow this https://www.ruby-lang.org/en/installation/
1. Install RVM using from http://rvm.io/rvm/install
2. use default ruby as 2.1.1
3. install Rails 4 http://railsapps.github.io/installrubyonrails-ubuntu.html

    #Technology Stack

    1. Bootstarp

    2. AngularJS

    3. Rails 4

    4. Ruby 2.1.1

    5. MongoDB
